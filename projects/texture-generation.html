Texture Generation Generative Adversarial Network

Python, Tensorflow

The aim of this project was to use deep learning to generate textures for games from a given sample. The final result is a GAN network that I built in Python using Tensorflow that can generate a new texture with a given dataset. This project is part of the Advanced technologies module during my third year at University and is documented in 5 part short series, the result of which can be seen here.
